# Sudoku

![Sudoku grid](Capture.PNG)

## Overview
Sudoku is a python project that generates sudoku's grid and allow to play on them. The project uses pygame library for the UI.

## Features

- Generates grids based on given difficulty

- UI to play on grid


## Requirements

- pygame
